Single Select and Multi select

A way for the user to select items from a list.

Description

When users need to select several options from a list they can do it by selecting one option at a time and get an overview of what that has been selected. All users do not understand the standard HTML multiple select where keyboard combinations are required to do multiple select and it is hard to get an overview. When users use a longer list for single or multi select they need to be able to search in the list for quicker and easier interaction. To avoid different appearances across applications it is recommended to use the component for every single or multiple select.

Single select

It is recommended to use deselect on all single selects. This allows the users to easily deselect the option that they have chosen.

The recommended threshold to get a search field in the select box is 10 items.

If there are less than 5 items in the list a radio button group is preferred.

Multi select

If more than one row of items are selected the box should grow vertically.

If there are less than 5 items in the list a check box group is preferred.

Component

Select2 is a component meant to be used for searching in large amount of data with the use of dropdowns. The component makes it possible to search for result either by group or by the name of the content. If a search would return no result this would be displayed in the view and an appropriate text should be visible.
